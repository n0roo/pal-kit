# PAL Kit TUI 키맵 가이드

> vim-style 키보드 단축키 레퍼런스

---

## 전역 키맵

모든 화면에서 사용 가능한 단축키입니다.

| 키 | 동작 | 설명 |
|----|------|------|
| `q` | 종료 | TUI 종료 |
| `Ctrl+C` | 강제 종료 | 즉시 종료 |
| `?` | 도움말 | 도움말 오버레이 토글 |
| `r` | 새로고침 | 데이터 갱신 |
| `Tab` | 다음 탭 | 탭 순환 (다음) |
| `Shift+Tab` | 이전 탭 | 탭 순환 (이전) |
| `1` | Status 탭 | Overview 화면 |
| `2` | Sessions 탭 | 세션 목록 |
| `3` | Ports 탭 | 포트 목록 |
| `4` | Projects 탭 | 프로젝트 목록 |
| `5` | History 탭 | 이벤트 히스토리 |
| `6` | Workflows 탭 | 워크플로우 목록 |
| `7` | Docs 탭 | 문서 목록 |
| `8` | Conventions 탭 | 컨벤션 목록 |
| `9` | Agents 탭 | 에이전트 목록 |

---

## 리스트 탐색

목록 화면에서 사용하는 단축키입니다.

| 키 | 동작 | 설명 |
|----|------|------|
| `j` / `↓` | 다음 항목 | 커서 아래로 |
| `k` / `↑` | 이전 항목 | 커서 위로 |
| `g` | 첫 항목 | 목록 맨 위로 |
| `G` | 마지막 항목 | 목록 맨 아래로 |
| `Ctrl+d` | 반 페이지 아래 | 빠른 스크롤 |
| `Ctrl+u` | 반 페이지 위 | 빠른 스크롤 |
| `Enter` | 선택/상세 | 항목 선택 또는 상세 보기 |
| `/` | 검색 | 필터 모드 진입 |
| `Esc` | 검색 취소 | 필터 초기화 |

## 패널 전환

분할 화면에서 패널 간 이동입니다.

| 키 | 동작 | 설명 |
|----|------|------|
| `h` / `←` | 왼쪽 패널 | 목록 패널로 이동 |

## 커스터마이징 (향후)

키맵은 `~/.pal/config.yaml`에서 커스터마이징 가능합니다 (향후 지원 예정):

```yaml
tui:
  keymap:
    quit: "q"
    help: "?"
    next_item: ["j", "down"]
    prev_item: ["k", "up"]
    # ...
```
